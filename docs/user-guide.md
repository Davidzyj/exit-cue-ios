# Exit Cue User Guide

## What the app does

Exit Cue is a local iPhone app for setting timed call-style reminders. It is intended for moments where a user wants a calm, private prompt to leave an uncomfortable date, meeting, or social setting.

Exit Cue does not make real calls, receive calls, use CallKit, or copy the iOS Phone app.

## Start a cue

1. Open Home.
2. Select a caller profile.
3. Select a delay, or choose a custom minute value.
4. Tap Start.
5. Home shows the active countdown.
6. When the countdown finishes, the cue screen appears.

## Immediate cue

Tap Now on Home to present the cue immediately.

## Answer or dismiss

- Answer opens the scripted cue screen.
- End and mark safe records the cue as completed.
- Dismiss records the cue as dismissed.
- Cancel on Home stops an active cue and records it as canceled.

## Caller profiles

The app includes built-in neutral profiles and lets users create custom profiles. Custom profile fields are saved only on the device.

Editable fields:
- Name
- Label
- Cue line
- Accent color

## History

History shows recent completed, dismissed, and canceled cues. Users can clear history at any time.

## Settings

Settings includes:
- Language: System, English, Simplified Chinese, Japanese
- Sound
- Haptics
- Optional local notifications
- Privacy Policy link
- Support link
- Version

The settings screen does not show bundle ID, raw URLs, or the support email address.

## Privacy

No account is required. No backend is used. No analytics, ads, or tracking SDKs are included. Local app data stays on the device unless the user deletes the app or clears history/custom profiles.
